Clon de Instagram
Python Django Mysql html css Bootstrap

Se realizó una aplicación similar a instagram utilizando el framework django. La aplicación consiste en publicaciones de fotografías hechas por usuarios. Los usuarios podrán registrarse, creando una cuenta, podrán iniciar sesión con su respectiva cuenta, podrán editar su perfil con una descripción de ellos si es que así lo quisieran, podrán postear fotografías, podrán obervar los posts que realizan los demás, así como tambien podrán visitar el perfil de los otros usuarios.
