Clon de Instagram
Python Django Mysql html css Bootstrap

Se realizó una aplicación similar a instagram utilizando el framework django. La aplicación consiste en publicaciones de fotografías hechas por usuarios. Los usuarios podrán registrarse, creando una cuenta, podrán iniciar sesión con su respectiva cuenta, podrán editar su perfil con una descripción de ellos si es que así lo quisieran, podrán postear fotografías, podrán obervar los posts que realizan los demás, así como tambien podrán visitar el perfil de los otros usuarios.

Producto Digital
html css

Este proyecto es una pagina web resultado de un curso de HTML y CSS que trata sobre la venta de un producto como jna caja para gatos. Incluye la descripcion del producto, las caracteristicas principales del producto, opiniones de clientes que adquirieron el producto, un formulario para los datos del cliente que va realizar la compra, además de estar conectado al sistema de pagos PAYPAL.

Anuncio de Conferencia
html css Bootstrap

Este proyecto es una pagina web resultado de un curso de BOOTSTRAP que trata sobre el anuncio de una Conferencia que se va a realizar en Hawai. Se hace la descripcion de la conferencia, se anuncia quienes van a ser los expositores con su respectivo tema del que hablaran, se describe el lugar donde se realizara la conferencia e incluye un formulario para que uno pueda hacer la solicitud para convertirse en un expositor mandando el tema del que hablará.
